# Sistema de Login com FAQ

## Descrição do Projeto

Este projeto consiste no desenvolvimento de uma página de login utilizando HTML5 e Bootstrap 5, integrada a uma página de Perguntas Frequentes (FAQ). A proposta da atividade foi aplicar conceitos básicos de desenvolvimento web, organização estrutural de páginas e utilização de Inteligência Artificial como ferramenta de apoio na construção do conteúdo e da documentação.

A página principal (index.html) apresenta um formulário de login contendo campo para inserção de e-mail, campo de senha, opção “Lembre de mim” e botão de acesso. Além disso, foi implementado um botão adicional que redireciona o usuário para a página de FAQ (faq.html), permitindo uma navegação simples e intuitiva.

A página de FAQ foi construída utilizando o componente Accordion do Bootstrap, organizando perguntas e respostas em formato expansível (abre e fecha), melhorando a experiência do usuário.

## Funcionalidades

- Tela de login com campo de e-mail e senha  
- Opção "Lembre de mim"  
- Botão de acesso  
- Botão para redirecionamento ao FAQ  
- Página de Perguntas Frequentes interativa  
- Botão para retornar à tela de login

## Como Executar o Projeto

1. Baixe ou clone o projeto.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Utilize o botão **"Ir para FAQ"** para acessar a página de perguntas frequentes.

Não é necessário instalar dependências adicionais.
